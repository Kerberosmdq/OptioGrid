# OptioGrid

OptioGrid is a smart comparison tool that helps you organize, analyze, and choose the best options for your purchases.

## Features
- **Template System**: Create custom comparison templates for any category.
- **Smart Extraction**: Paste a link and automatically get details.
- **Visual Grid**: Compare items side-by-side with a clean UI.
- **AI Recommendations**: Get unbiased pros/cons and scoring.
- **Real-time Collaboration**: Invite family or friends to edit.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Node.js, Express.
- **Database**: Supabase (PostgreSQL).
- **AI**: LLM Integration.

## Getting Started
See [spec.md](./spec.md) for detailed technical specifications.

### Prerequisites
- Node.js 18+
- Supabase Account

### Installation
1. Clone the repo.
2. `cd frontend && npm install`
3. `cd backend && npm install`
4. Configure `.env` files.
5. Run `npm run dev`.
